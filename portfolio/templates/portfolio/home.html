eBay在庫管理Webアプリ

制作期間：2024.09 – 2025.07 公開：Heroku（現在停止） 開発終了

複数ECサイトに出品している商品の在庫管理を効率化するために開発した 個人用Webアプリケーション。 Djangoを用いた外部サービス連携や、Heroku Schedulerによる定期実行処理の設計を目的として実装。

概要

  • EC出品ページURLの登録・商品情報スクレイピング
  • 商品情報のGoogleスプレッドシート自動保存
  • eBay APIによる出品商品情報取得
  • ECサイト商品とeBay商品の紐づけ管理
  • 売り切れ時のeBay出品自動停止機能

使用技術

Python / Django / PostgreSQL（本番環境） / SQLite（開発環境） / Google Sheets API / eBay API / Heroku / Heroku Scheduler
